Why this matters

Picking the wrong license is expensive

Microsoft 365 has more than 30 different licenses. The right one depends on who the person is — an admin, an everyday user, a frontline worker, a teacher, a student, and so on. Picking the wrong one can easily cost 10× too much, or break a Microsoft rule about who's allowed to use it.

🎯

One license per type of user

An admin, a frontline shift worker, a teacher, and a guest from another company all need different Microsoft 365 licenses — sometimes none at all. The right answer can change the monthly cost by 10× or more.

🔒

Some licenses have rules about who can use them

Frontline, Education, Government, and Nonprofit licenses are only for people who actually qualify. Microsoft can audit you and ask you to prove it. Picking the wrong one isn't just a billing problem — it's a compliance problem.

📘

Every answer links to the source

Every recommendation links back to Microsoft Learn, the Microsoft Product Terms, and the m365maps.com comparison charts so you can double-check in seconds.

One person can need more than one license. The same person can have a regular account for email and Office and a separate admin account — those two accounts need different licenses. Just run the tool once for each.

Who this tool covers

Who you can pick a license for

The tool covers eight types of people, plus one warning for a setup Microsoft tells you to avoid. Don't see your situation? Let us know on GitHub.

🛡️ Full tree

Privileged (dedicated) admin

A separate cloud-only identity used only for privileged work — Global Admin, Identity / Security / Compliance / Intune / Teams / Exchange Admin. Entra ID Free until it crosses into PIM, ID Protection, Defender XDR, Purview E5, Intune Suite, Teams Premium, Entra Suite, Copilot, or Agent 365.

⚠️ Warning path

Primary daily-use account with admin roles

Same identity does mail / Teams / Office and privileged work. Microsoft does not recommend this. Always needs a base service license plus any premium tier the role triggers.

💼 Curated guidance

Information / knowledge worker

Office / Teams / SharePoint / OneDrive / Outlook user on a managed PC. M365 E3, M365 E5, M365 Business Premium (≤ 300 seats), + Copilot, + Teams Premium, + Intune Suite, + Entra Suite, M365 E7.

👷 Curated guidance

Frontline worker (F1 / F3)

Shift / deskless / field worker — retail, manufacturing, healthcare, hospitality, first responders, field engineers. Microsoft 365 F1 / F3 at a fraction of E-SKU price. Eligibility-gated under the Product Terms.

🎓 Curated guidance

Education — faculty / student

Qualifying academic institution only. Microsoft 365 A1 / A3 / A5 for faculty and students — different entitlements and pricing per role. A5 Compliance / A5 Security standalone bumps available.

🏛️ Curated guidance

Government — GCC / GCC High / DoD

US sovereign clouds. M365 G1 / G3 / G5 + F-tier Government. Confirm Copilot, Entra Suite, Defender XDR, Purview Premium availability — sovereign clouds lag commercial GA.

💝 Curated guidance

Nonprofit

Charitable org enrolled in Microsoft Nonprofits. Business Premium grant (up to 300 seats), Nonprofit Staff Pricing on Basic / Standard / Premium / E3 / E5 / F3. Eligibility re-validated annually.

🏢 Curated guidance

Small / mid-size business (≤ 300 seats)

Commercial org with no more than 300 seats. Microsoft 365 Business Basic / Standard / Premium. Hard 300-seat cap — above that you must move to Enterprise SKUs.

🔗 Curated guidance

External ID / B2B guest

Partner / vendor / contractor invited as a guest. Billed by Monthly Active Users (MAU) under Entra External ID — not per-seat. First 50 000 MAU free per tenant. Premium features (P1 / P2 / Verified ID) priced per MAU.
